--Descrição Geral

O sistema é uma aplicação web simples que permite gerir uma lista de nomes (alunos). Foi construído utilizando tecnologias puras (HTML5, CSS3 e JavaScript Vanilla), sem a dependência de frameworks externos, focando na manipulação de DOM e lógica de arrays.

--Funcionalidades Principais 


*Cadastro: Adiciona um nome ao array e atualiza o ecrã instantaneamente.


*Remoção: Remove o último nome da lista.


*Inversão: Inverte a ordem atual dos alunos cadastrados.


*Formatação Automática: Transforma o texto em maiúsculas assim que o utilizador termina de digitar.


*Tema Dinâmico: Alternância entre Modo Claro e Escuro com deteção automática do sistema.

--Recursos Técnicos Utilizados (Checklist de Requisitos) O projeto cumpre 100% das solicitações da imagem, conforme detalhado abaixo:


*Interface (Item 1): Criada com input, botões e div de exibição.


*Armazenamento (Item 2): Utiliza um Array global de alunos.


*Funções Obrigatórias (Item 3): Implementação das funções adicionarAluno, removerAluno, inverterLista e mostrarLista.


*Métodos de Array (Item 4): Uso de .push() para adicionar, .pop() para remover, .reverse() para inverter e .join() para formatar a exibição.


*Eventos (Item 5): Aplicação de onclick nos botões e onchange no campo de texto.


*Interação Inicial (Item 6): Uso de window.onload com prompt para capturar o nome do utilizador e exibir mensagens de boas-vindas.


*DOM (Item 7): Manipulação completa via getElementById, alteração de innerHTML, captura de value e alteração de style.


*Validação (Item 8): Bloqueio de inserção de valores vazios com um alerta de erro.
